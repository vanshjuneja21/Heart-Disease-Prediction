# Heart-Disease-Prediction
**Machine Learning project using Logistic Regression**
Overview:
This Project predicts whether a person has heart disease using Machine Learning Techniques

Models Used:
* Logistic Regression
  
Dataset
* UCI Heart Disease Dataset

Features:
* Data preprocessing (handling categorical variables)
* encoding
* Model training and evaluation
* Accuracy, Confusion Matrix, Classification Report

Results:
* Accuracy: 84.8%

Key Learnings:
* Importance of data preprocessing
* Handling categorical variables

Tech Stack:
* Python
* Pandas, NumPy
* Scikit-learn

Sample Prediction
* Input: Patient data
* Output: Heart Disease/ No Heart Disease
